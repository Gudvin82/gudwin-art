# One-Minute Demo Script

## Goal

Show product value in 60 seconds for a recruiter, partner, or investor.

## Script

1. Open homepage and switch era cards to show the concept breadth.
2. Go to `/create` and upload one photo.
3. Adjust crop quickly to show user control.
4. Pick one era and style preset.
5. Start generation and show progress UX.
6. Show paywall step (`Telegram Stars`) and HD unlock behavior.
7. Finish with account history page to show repeatable user journey.

## Talking Points

- Productized AI flow, not just static generation.
- UX-first funnel with monetization path.
- Era-based differentiation and clear portfolio positioning.
- Public showcase is safe: no production secrets or private core logic.
