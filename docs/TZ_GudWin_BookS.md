# ТЗ (короткая рабочая версия) - GudWin BookS

## 1. Продукт

- Название: **GudWin BookS / ГудВин Букс**.
- Формат: веб-платформа для генерации:
  - сказок (текст + иллюстрации + озвучка),
  - комиксов (сценарий + страницы/кадры + реплики).
- Вход: только через Telegram.
- Оплата: только Telegram Stars (XTR).

## 2. UX и стиль

- Визуал: арт + комиксы + старинные книги + сказочная эстетика.
- Главная:
  - splash-заставка 3-5 секунд,
  - блок возможностей,
  - 3 шага работы,
  - тарифы,
  - отзывы/кейсы,
  - подвал с документами.
- Обязательные анимации:
  - reveal on scroll,
  - деликатные частицы/звезды,
  - glow-hover кнопок,
  - перелистывание секций.

## 3. Функционал ЛК

- Разделы:
  - Мои сказки,
  - Мои комиксы,
  - Создать новое,
  - Тариф/звезды,
  - Настройки.
- Генерация сказок:
  - возраст, тема, мораль, длительность, стиль,
  - редактирование текста,
  - перегенерация иллюстраций.
- Генерация комиксов:
  - жанр, синопсис, число страниц, стиль,
  - редактирование реплик,
  - перегенерация отдельных кадров.
- Озвучка:
  - выбор голоса,
  - встроенный плеер,
  - скачивание по тарифу.

## 4. Монетизация

- Тарифы:
  - Бесплатный,
  - GudWin Light,
  - GudWin Pro.
- Все платные операции списывают Stars.
- Отдельный биллинг-сервис и учет кредитов пользователя.

## 5. Архитектура

- Frontend: React/Vite SPA (текущий MVP).
- Backend (обязательно для запуска):
  - Telegram auth verify,
  - billing/stars ledger,
  - generation queue (text/image/tts),
  - project storage.
- Интеграции:
  - Telegram Login/WebApp,
  - Telegram Stars,
  - LLM + image model + TTS.

## 6. Право и комплаенс (РФ 2026)

- RU язык по умолчанию, EN как дубль.
- Обязательные страницы:
  - оферта,
  - политика конфиденциальности,
  - политика ПДн (152-ФЗ),
  - согласие на обработку ПДн,
  - правила ИИ-контента,
  - реквизиты владельца.

## 7. Безопасность запуска (must-have)

- Server-side верификация Telegram подписи.
- HttpOnly сессии + CSRF-защита.
- RBAC/ownership проверки.
- Signed URLs для загрузок.
- Rate limiting + WAF + audit logs.
- Валидация всех входных данных.
- Антиабьюз фильтры промптов/контента.
